# Liberty Arm

Liberty Arm est un **site web fictif** conçu dans le cadre d’un **projet Roleplay** pour **Garry’s Mod**, destiné aux serveurs de type **GTA City RP**.

Le site simule une plateforme de **vente d’armes RP**, utilisée exclusivement comme support narratif et immersif en jeu.

---

## Objectif du projet

L’objectif de Liberty Arm est de fournir une **interface web crédible et réaliste** permettant :

* d’enrichir l’immersion Roleplay,
* de servir de support à des scènes RP (achats, contacts, enquêtes, événements),
* de renforcer la cohérence de l’univers GTA City RP.

Le projet n’a **aucun lien avec la réalité** et ne propose **aucune fonctionnalité réelle de vente ou d’achat**.

---

## Contexte d’utilisation

* **Plateforme** : Garry’s Mod
* **Type de serveur** : GTA City RP
* **Usage** : Strictement Roleplay
* **Public cible** : Joueurs et équipes RP
